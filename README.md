# DataScience-week-app

Tutorial for improve skills: 'Semana de DataScience' (Minerando Dados) by Marcus Mariano

## Required Files
setup.sh
Procfile
pytroject.toml

---

## How to Install

Commands for Terminal
```sh

poetry install

```

---

## How to Run

Commands for Terminal
```sh

poetry run streamlit run app.py

```
